## 📖 遊戲背景
下午 14:00，一場突如其來的雷雨擊毀了都市電網。身為市長，你手中的緊急電力僅剩 **100%**。

本遊戲將帶領學生經歷兩個截然不同的決策層次：
1. **第一階段：守護生命底線 (SOP)** - 考驗基礎防災知識。面對醫院救災、化災處理與地震避難，是否有專業的判斷力？
2. **第二階段：資源權衡兩難 (Trade-off)** - 步入資源枯竭的深水區。當電力只夠救一邊時，你要保護「民生弱勢」還是「城市運作系統」？

## 🌟 核心教學價值
本遊戲結合 2026 數位教育實驗計畫，引導學生思考：
* **專業程序 (SOP)**：理解在災害發生初期，遵循科學與專業流程是降低損害的唯一途徑。
* **機會成本與權衡**：真實體會資源有限下，任何「獲得」都伴隨著「犧牲」。
* **價值觀形塑**：透過最後的「市長人格分析」，讓學生反思自己的決策偏好——是傾向大局效能，還是人道關懷？

## 🛠️ 技術規格
* **題庫規模**：內建 **102 題** 海量題庫（50題 SOP 題 + 52題 決策兩難題）。
* **遊戲機制**：每場演練由系統隨機抽題，確保每次遊玩都有不同的情境組合。
* **分析系統**：即時回饋決策後果，並根據選擇邏輯自動生成「市長性格分析報告」。
* **純淨架構**：原生 HTML5 / CSS3 / JavaScript 開發，無須安裝依賴，開啟網頁即可教學。
